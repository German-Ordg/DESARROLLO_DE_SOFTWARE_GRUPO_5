12.1. Facturacción

Menú desplegable tipo de pago

En este control se selecciona el tipo de pago que va a realizar el cliente.
Arriba

Botones Tipo de Factura

Aquí se selecciona el tipo de factura ya sea con nombre o sin nombre dependiendo de lo seleccinado se mostrarán o no los campos de RTN y el campo de Nombre del cliente.
Arriba

Caja de texto RTN

En este campo se ingresa el RTN del cliente.
Arriba

Caja de texto Nombre Cliente

En este campo se ingresa el nombre del cliente .
Arriba

Caja de texto Dirección

En este campo se ingresa la dirección a la que será enviado el paquete.
Arriba

Botón minimizar

Al seleccionar este botón se minimiza la pantalla del sistema.
Arriba

Botón Regresar

Al seleccionarlo se regresa al menú principal.
Arriba

Botón nueva factura

Al seleccionarlo recarga la pantalla y limpia todos los datos para ingresar una nueva venta.
Arriba

Caja de texto Código de producto

En este campo se ingresa el código del producto del que se quieren obtener los datos.
Arriba

Botón Buscar

Al ingresar un código se presiona este botón y cargará los datos del producto en las otras cajas de texto.
Arriba

Contenedor cantidad

En este campo se ingresa la cantidad del producto que se quiere vender.
Arriba

Botón Eliminar

Este botón funciona para eliminar un producto de la tabla inferior que es donde se almacenan temporalmente los datos de la venta actual, para lo cual se debe seleccionar un item de dicha tabla.
Arriba

Botón Actualizar

Este botón funciona al editar un producto de la tabla inferior en caso que se quiera cambiar la cantidad de producto vendido, al cambiar dicha cantidad se da click izquierdo en este botón y actualiza los datos en la tabla.
Arriba

Botón Editar

Este botón funciona para editar la cantidad de un producto de la tabla temporal inferior, luego de editar dicha cantidad en el contenedor de cantidad (11) se tiene que dar click en el botón de actualizar y se modificarán los datos.
Arriba

Botón Agregar

Este botón sirve para agregar un producto que se haya buscado previamente al detalle de la venta actual la cual aparece en la tabla inferior.
Arriba

Botón Calcular Factura

Este botón sirve para calcular el impuesto y el total a pagar, al momento de estar añadidos todos los productos que ordenará el cliente, se selecciona y aparece en las cajas de texto los datos del total a pagar.
Arriba

Botón Imprimir Factura

Este botón sirve para concretar la venta, una vez estando seguro que la venta se hizo se selecciona este botón y pasa a la parte de la factura. en caso de no aparecer habilitado presione el botón de calcular factura (16) para habilitarlo.
Arriba
