11.1. Productos

 

Seleccionador de busqueda

Permite Seleccionar el método o forma de busqueda del producto a registrar, ya sea por Código o por categoría.
 
Arriba

Código de producto

Campo editable para colocar el código del producto a registrar.
 
Arriba

Cuadro de busqueda

Permite ya sea por código o categoría, realizar la busqueda del producto a registrar.
 
Arriba

Botón de selección

Permite realizar la selección de la busqueda del cuadro de búsqueda.
 
Arriba

Campo de compra (Und.)

Permite describir el precio de compra del producto únicamente por unidad en la compra.
 
Arriba

Campo precio venta (Und.)

Permite describir el precio de venta del producto únicamente por unidad en la compra.
 
Arriba

Campo de descripción

Campo que permite escribir una descripción para el producto que se registrará en la compra.
 
Arriba

Campo Talla

Campo que permite definir talla del producto en caso de tratarse de ropa o calzado en la compra.
 
Arriba

Campo de categoría

Permite describir la categoría perteneciente al producto en la compra.
 
Arriba

Campo de cantidad

Permite definir una cantidad de del producto a registrar en la compra.
 
Arriba

Campo de descuento

Permite describir el descuento que se otorgó al producto dentro de la compra.
 
Arriba

Botón Agregar

Botón que permite agregar un producto al registro de compra.
 
Arriba

Botón Terminar compra

Botón que permite finalizar y registrar en su totalidad la compra que se desea ingresar al sistema.
 
Arriba

Botón Cancelar Compra

Botón que permite regresar a la pantalla de Compras en caso de desear cancelar el registro de la actual compra.
